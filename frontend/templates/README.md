# Template Archive

This folder contains archived template files that are preserved for reference but not used in the current build.

## Files

### `alpinejs-version.html`
- **Original Location:** `frontend/public/index.html`
- **Description:** Complete Alpine.js implementation of the Trendsetter Resume Analyzer
- **Features:**
  - Alpine.js reactive state management
  - Tailwind CSS via CDN
  - File upload functionality
  - Real-time job description analysis
  - ATS compatibility scoring
  - Keyword matching
- **Date Archived:** January 2026
- **Reason:** Switching to Next.js React-based `app/page.tsx` implementation
- **Status:** Fully functional standalone HTML file
- **Use Case:** Can be used as a reference for future vanilla JS/Alpine.js projects or as a lightweight alternative

## How to Use These Templates

These files are not part of the build process. They are kept for:
- Reference and documentation
- Alternative implementation approaches
- Future project templates
- Design system reference

To use any of these templates, copy them to a new project or reference their code patterns.
